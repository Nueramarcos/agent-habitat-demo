# Contributing to Nueramarcos/agent-habitat-demo

## Development Setup

1. **Clone the Repository**
   ```sh
   git clone https://github.com/Nueramarcos/agent-habitat-demo.git
   cd agent-habitat-demo
   ```

2. **Install Dependencies**
   ```sh
   pip install -r requirements.txt
   ```

3. **Run Tests**
   ```sh
   pytest
   ```
